Welcome player!

Think fast... SIMON says, "Chase my flashing lights and sounds"!


The challenge is to repeat the ever-increasing random signals that SIMON generates.

How To Play?


1. Turn the power switch on.
2. Press START button; It will now give the first signal.
3. Repeat the signal by pressing the same color pad.
4. It will duplicate the signal and add one.
5. Repeat those 2 signals by pressing the same color pad.
6. Continue doing this till you reach as high as possible.

Lets see how high you can get!

Strict Mode


When Strict mode is selected you cannot get your sequence wrong at any point or your game will be reset back to the start
